Tim Berners-Lee

Inženýr, tvůrce World Wide Webu a propagátor otevřeného webu.

Krátký životopis

Sir Timothy John Berners-Lee (nar. 8. června 1955) je britský počítačový vědec, který v 80. a 90. letech vytvořil základní technologie World Wide Webu: URL, HTTP a HTML. Později založil a vedl W3C (World Wide Web Consortium) a působí jako obhájce otevřeného a demokratického webu.

Hlavní přínosy

  • Vynález World Wide Webu (hypertext propojený přes internet).
  • Definice klíčových protokolů a jazyků: HTTP, HTML, URL.
  • Založení W3C — organizace pro standardizaci webu.
  • Propagace otevřenosti, interoperability a ochrany soukromí na webu.

Krátké časové body

Rok / Období Událost
1955 narození v Londýně.
1989–1990 návrh a první implementace WWW v CERNu.
1994 založení W3C (World Wide Web Consortium).
